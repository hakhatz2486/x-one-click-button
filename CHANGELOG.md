# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.1] - 2026-07-19

### Changed

- Configure the Greasy Fork script page and update URLs in the metadata

## [1.0.0] - 2026-07-19

### Added

- One-click `Mute` and `Block` buttons on posts
- An `Unfollow` button displayed alongside the native following button on following lists
- `Unfollow` and `Block` buttons for removing followers or blocking accounts on follower lists
- Support for Japanese and English X menus
- Dynamic button injection that follows X page navigation and list rendering
- Safeguards against inverse-action matches, stale menus, and concurrent actions
